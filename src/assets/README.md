# Assets Directory

Place your app icons and images here.

## Recommended Structure

- `logo.png` - App logo
- `icons/` - Icon files
  - `check.png`
  - `delete.png`
  - `edit.png`
  - etc.

## Icon Resources

You can get free icons from:
- [Ionicons](https://ionic.io/ionicons)
- [Material Icons](https://fonts.google.com/icons)
- [Feather Icons](https://feathericons.com/)

For React Native, consider using icon libraries:
- `react-native-vector-icons`
- `@expo/vector-icons` (if using Expo)
